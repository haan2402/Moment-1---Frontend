Arbetsprocess

För att skapa HTML, CSS och Javascript använder jag mig av Visual Studio Code. Node.js har använts för att installera och använda npm paket och för att automatisera.

Har gjort en mappstruktur där all kod som görs i VSC är lagd i mappen src, och har gjort en gitignore-fil där jag tagit bort node_modules, package-lock.json, dist och parcel-cache för dessa vill jag inte versionhantera.

Jag gör regelbundna commits i min development-branch, som jag sedan merge ihop med main-branch för publicering. Har gjort ett Netlify-konto som jag har kopplat till min Github för att publicera min webbplats.

Automatisering och publicering

  • Parcel: För att minifiera HTML, CSS och Javascript kod. Bilderna på webbplatsen optimeras till WebP som är ett modernt filformat.
  • Publicering: Webbplatsen publiceras med Netlify
